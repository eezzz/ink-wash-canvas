# 水墨流彩 · Ink Wash Canvas

基于 WebGL2 的双介质数字绘画画布：**水墨**（毛笔书写 + 流体晕染）与**油画**（鬃毛厚涂 + 湿料调和）。
单文件应用，零依赖，浏览器打开即用。

- 在线版：https://eezzz.github.io/ink-wash-canvas/
- 本地版：直接打开 `ink-wash-canvas.html`（需支持 WebGL2 浮点渲染的现代浏览器）

---

## 功能

### 水墨介质
- **水墨毛笔**（笔画式书写引擎）
  - 完整笔锋生命周期：锋尖着纸（细）→ 铺毫/行笔（粗）→ 提笔相"边走边提写出锋"（细）
  - 起笔：轻触即走 = 露锋尖入；按住铺毫再走 = 顿笔满幅起（切入笔头，锋角留在着纸点）
  - 收笔：抬笔速度决定锋形——快提悬针长锋、慢提丰圆、停顿后抬笔顿收圆头
  - 收锋方向追踪：抬笔前手的挑/按决定锋从上缘/中线/下缘出
  - 墨量系统：行笔耗墨渐淡，枯笔现飞白（笔画两侧断续，笔芯不分叉）
  - 速度笔宽：快细慢粗，非对称包络防收笔鼓包；提按弹性回弹
- **晕染**（GPU 流体模拟）：点染墨晕动态绽开、拖染随水流，颜色如真实颜料减色混合（Beer–Lambert 吸光模型，花青+藤黄=绿）

### 油画介质
- 专用笔触引擎：扁头轮廓、鬃毛沟槽厚度纹、破碎撕裂边缘
- 厚涂浮雕：按颜料堆积厚度实时光照（棱顶高光、沟底沉影、油光反射）
- **粘稠滑杆**：厚料完全覆盖底色 ↔ 稀料蘸起底下湿彩就地调和（含笔尖蘸色机制）
- 湿料推挤：拖笔把已有湿颜料黏稠地刮开混色
- 钛白覆盖提亮 / 低浓度透明罩染

### 通用控制
- 调色盘：钛白 + 墨 + 八种国画色
- 滑杆：笔锋粗细（发丝级到阔笔）、浓淡（水墨墨色 / 油画铺料量）、水分（湿润洇边↔枯笔飞白）、粘稠（油画覆盖↔调和）、防抖（笔尖位置低通滤波）
- 撤销 / 重做（8 步 GPU 快照，Ctrl+Z / Ctrl+Shift+Z / Ctrl+Y）
- 存画 / 取画（IndexedDB，含介质记忆）、照片导出 PNG、洗砚清空
- 控制面板可收起（手机小屏默认收起，整屏作画）

## 技术栈

单文件 HTML + 原生 JavaScript + WebGL2，无任何运行时依赖。

| 模块 | 技术 |
|---|---|
| 流体模拟 | Stable Fluids（半拉格朗日平流 + Jacobi 压力求解 + 涡量约束），RG16F/RGBA16F 半浮点 FBO 乒乓 |
| 水墨显色 | 染料存吸光度，`paper × exp(-absorbance)` 减色混合（Beer–Lambert） |
| 毛笔笔触 | 逐笔 stamp 到独立笔画层（max 合成防叠加发糊），抬笔合入画纸；提笔相为时间驱动的物理过程 |
| 油画笔触 | 颜料场存（预乘颜色, 厚度），覆盖式合成 + 蘸色调和，厚度场差分法线做浮雕光照 |
| 防糊化 | 平流速度下限（velFloor）：静止区域不重采样，笔触纹理永久锐利 |
| 存储 | IndexedDB 存 Blob（PNG 双层编码：上半颜色/下半厚度，sqrt 量化）；localStorage 存偏好 |
| 输入 | Pointer Events（含 coalesced 高频事件、压感、合成事件容错） |
| 测试 | `test/run-tests.ps1`：语法检查 → 桩 DOM/WebGL 冒烟 → 浏览器自测（`?selftest` / `?selftest=oil` 页面内合成事件自动书写）截屏出图 |
| 调试 | `?debug` 实时事件轨迹层；页面左上角构建号 |
| 部署 | GitHub Pages，push 即发布 |
